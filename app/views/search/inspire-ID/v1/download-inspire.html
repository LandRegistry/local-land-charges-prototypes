Local land charges spatial data (INSPIRE)

Data showing the indicative extent and position of local land charges (categorised as being within INSPIRE theme groups) in England and Wales.

Not every local authority dataset has been migrated to HM Land Registry. As more are migrated, more will appear in the list below.

This dataset was published on 6 March 2022. You can access the data until we update the files. New files are published on the first Sunday of every month.

You can:

  • view all polygons on a map
  • download polygons by local authoritylocation

The dataset

This is an open-source dataset, developed to follow the INSPIRE Directive.

The dataset uses spatial data from HM Land Registryâ€™s local land charges register. It shows the indicative locations of local land charges in England and Wales which are part of a category group in the Inspire themes register. Note that migrating local land charges data from local authorities to HMLR is an ongoing process, so not every local authority area is presently available on this service.

About INSPIRE polygons

Each polygon has a unique identification number called an INSPIRE ID. You can use an INSPIRE ID to Search for a local land charge.

Each local land charge will have a polygon associated with it and may have more than one.

Polygons divide into separate data files for each local authority area. If a polygon is on the boundary between local authorities, it will appear in both data files.

To find out the definitive extent of land in a local land charge, you need to inspect the source documents. The originating authority hold these.

Access the data

The dataset updates on the first Sunday of each month and shows data from the previous month. For example, data published in April will contain data from March.

You will be able to access the data until we publish new data the following month.

The files are published as GML files. You will need a Geographical Information System (GIS) to view downloaded polygons.

GIS will enable you to store or edit downloaded data. You can then compare or integrate with other available datasets.

Read the guidance on using GML format files.
