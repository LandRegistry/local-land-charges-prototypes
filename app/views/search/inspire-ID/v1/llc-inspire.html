The dataset

This is an open-source dataset, developed to follow the INSPIRE Directive.

The dataset uses spatial data from HM Land Registryâ€™s local land charges register. It shows the indicative locations of local land charges in England and Wales. These charges form part of a category group in the INSPIRE themes register.

INSPIRE themes

Area management, Restrictions, Regulation zones and Reporting units

This theme covers areas under an environmental policy or activity impacting the environment.

Policies include:

  • reducing pollution levels
  • conserving natural resources
  • protecting plant, animal, and human health
  • controlling development

Local land charges which form part of this theme group include smoke control orders.

Land use

This theme covers the use and functions of a territory and its description of land. This depends on its socio-economic and ecological purpose.

The 2 types of land use are:

  • The existing land use, which outlines the use and functions as they are now
  • The planned land use, defined by spatial planning authorities to outline ways to use the land

Local land charges which form part of this theme group include Legal agreements as to the use of land, and Tree Preservation Orders.

Protected sites

This theme focuses on the specific conservation of designated areas of land or sea. It protects and maintains:

  • listed buildings
  • biological diversity
  • conservation areas

Local land charges which form part of these theme group include Ancient Monuments and Sites of Special Scientific Interest.

About INSPIRE polygons

Each polygon has a unique identification number called an INSPIRE ID. You can use an INSPIRE ID to Search for a local land charge.

Each local land charge will have a polygon associated with it and may have more than one.

Polygons divide into separate data files for each local authority area. If a polygon is on the boundary between local authorities, it will appear in both data files.

To find out the definitive extent of land in a local land charge, you need to inspect the source documents. The originating authority will hold these.

How you can use the data

Conditions of use

Your use of the INSPIRE local land charges spatial data polygons service is governed by conditions.

This dataset is available for use and reuse under the Open Government Licence (OGL). This licence enables public bodies to make their data available free of charge for reuse.

Use under the OGL is free. If you fail to comply with any of the conditions of the OGL, then the rights granted to you under the licence will end automatically.

Under the OGL, when reusing the data, you must acknowledge the source of the data. and include the following attribution statement:

You must provide a link to the following conditions: https://use-land-property-data.service.gov.uk/datasets/inspire/#conditions.

Access the data

The dataset updates on the sixth of each month and shows data from the previous month. For example, data published in April will contain data from March.

You will be able to access the data until we publish new data the following month.

The files are published as GML files. You will need a Geographical Information System (GIS) to view downloaded polygons.

GIS will enable you to store or edit downloaded data. You can then compare or integrate with other available datasets.

Read the guidance on using GML format files.
