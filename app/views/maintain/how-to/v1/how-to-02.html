Draw a charge extent area

  1. Select Add area and choose a location on the map to start drawing. Make sure the ends of the shape meet.
    • Draw curves by holding down the SHIFT key.
  2. Select Confirm. Make sure you capture all properties and land you want to search. This includes driveways, alleys, or gardens.

Edit a charge extent area

  1. Select Edit area. The charge extent area outline will highlight. The charge extent area background will blur.
  2. Drag the outline to edit.
  3. Select Edit area to finish.
  4. To delete a point in this mode:
    • hold down the ALT or OPTION key
    • select the point you want to delete with the left mouse button

Cut out an area from a charge extent area

  1. Select Cut out area. The charge extent area outline will highlight.
  2. Choose a place inside your area and start drawing.
  3. Make sure the ends of the shape meet.
Video guidance: Cut out an area from a charge area

Copy a building from the map

  1. Select Copy from map.
  2. Select the building you want to copy. This will highlight it. Using this feature once on a property may not always be enough to capture the whole affected area.
  3. Select Confirm. Make sure you capture all properties and land you want to search. This includes driveways, alleys, or gardens.
You can only copy from the map when you zoom in to a location. You should be able to see the outlines of buildings.

Draw a shape with Snap to map

  1. Select Add area.
  2. Select Snap to map.
  3. Hover near the line or point you want to snap to. The cursor will jump to that point automatically. You can also use this feature during other map interactions, like editing.
You can only use Snap to map when you zoom in to a location.

Delete one charge extent area

  1. Select Delete area. This will highlight your search area red.
  2. Select the search area you want to delete.
