Tools to create an extent

Copy from map

  1. Select Copy from map.
  2. Select the area you want to copy. This will highlight and act as the extent. You can select multiple features on the map to create one big extent. This includes garages, gardens or buildings.
You can only copy from the map when you zoom in to a location. You should be able to see the outlines of buildings.

Draw an extent area

  1. Select Add area and choose a location on the map to start drawing. Make sure the ends of the shape meet.
    • To draw curves hold down the SHIFT key.
    • You can also use Snap to map to trace existing lines on the map.
    • Hover near the line or point you want to trace. The cursor will jump to that point automatically.
  2. Select Confirm.
You can only use Snap to map when you zoom in to a location.
Video guidance: Draw an extent area
Video to go here

Edit your extent area

  1. Select Edit area. The extent area will highlight.
  2. Select and drag the outline of the area to edit it.

    3. To delete a point in this mode:

    • hold down the ALT or OPTION key
    • select the point you want to delete with the left mouse button
  3. Select Edit area to finish.
Video guidance: Edit your extent area
Video to go here

Draw a circle

  1. Select Add circle.
  2. Select a location on the map and start dragging the point until the circle covers the area you want.
Video guidance: Draw a circle
Video to go here

Delete one extent area

  1. Select Delete area. This will highlight the extent area.
  2. Select the extent area you want to delete.
Video guidance: Delete one extent area
Video to go here

Delete extent areas

  1. Select Clear all.
  2. Select Delete all. This will delete everything you have drawn on the map.
Video guidance: Delete an extent areas
Video to go here

Undo your last map interaction

  1. Select Undo. Your last interactions with the map will be undone.

You can select Undo as many times as you want.
